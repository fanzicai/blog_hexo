git clone项目时，采用ssh协议时，一般远端均是22端口。部署docker时，考虑端口映射成其他，如2222。
<!-- more -->

- 22端口
```
git clone git@hostname:repos/test.git
```

- 非22端口
```
git clone ssh://git@hostname:2222/repos/test.git
```

- 修改默认端口
> 编辑~/.ssh/config
> ```
Host ***
    Hostname ***
    User ***
    Port 2222
    IdentifyFile ~/.ssh/***
```
